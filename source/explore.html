Explore

いわゆる「アーカイブ」とか「サイトマップ」とか、本でいえば「索引」のような役割を果たすページです。任意キーワードによるサイト内検索機能や、コンテンツ種別・年別・タグ別に記事を分類したページへの入り口を提供しています。

{% include cse.html %}

コンテンツ種別で一覧

各コンテンツの新着記事5件のリストと、全記事一覧へのリンクです。各コンテンツのトップページと何が違うんだと問われれば…えーと、両方の新着をざっと知りたい時とかに役に立つかもしれない。

{% assign archives = site.archives | where: 'type', 'category' %} {% for archive in archives %}{% assign cat_name = archive.title | to_category_name %}

{{ cat_name }} 全記事

{% assign posts = archive.posts %}
    {% for post in posts limit:5 %}
  • {{ post.title }}
    • {% endfor %}
{% endfor %}

年（月別）で一覧

年別アーカイブページのリストです。新しめの Web ブラウザをお使いで、ウィンドウ幅が一定以上ある場合には、その年の月見出し（月でセクション分けしてある）に直接スキップするためのリストも表示されます。

{% assign year_posts = site.posts | group_by_year %}
    {% for year_post in year_posts %}{% assign year = year_post[0] %}
  • {{ year }}年 {% assign month_posts = year_post[1] | group_by_month %}
      {% for month in (1..12) reversed %}{% assign size = month_posts[month] | size %} {% if size != 0 %}{% endif %}{{ month }}月{% if size != 0 %}{% endif %} {% endfor %}
    • {% endfor %}

よく使われるタグから一覧

すべてのタグの中から記事が5件以上あるタグをピックアップして、記事数順に並べ替えたリストです。（記事が1件しかないようなタグも含めて）すべてのタグから探したい場合には「タグ一覧」をご利用下さい。

{% assign selected = site.archives | where: 'type', 'tag' | sort_by_popular %}
    {% for item in selected %}{% if item.posts.size < 5 %}{% break %}{% endif %}
  • {{ item.title }} ({{ item.posts.size }})
    • {% endfor %}
