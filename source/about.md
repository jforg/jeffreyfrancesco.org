概略 {#Outline}
--------------------

JeffreyFrancesco.org はその名の通り、主に北近畿（というよりは京都府北部限定）を中心に、ハモンドオルガン & 鍵盤ハーモニカ奏者・インターネットラジオ MC・Web 制作者 etc. として細々と活動を続けている、Jeffrey Francesco (JF) という名前の個人が運営する Web サイトです。

元はといえば Live や各種 Event 出演のお知らせなど、アマチュアミュージシャンとしての活動報告のために立ち上げたサイトなのですが、次第に本職である Web 制作関連の雑記を主に書き連ねた Weblog の方がメインコンテンツとなりつつありまして、要するに私レベルの演奏者の現実なんてこんなもんだ、ということがよく分かるサイトとなっております。涙

がんばって更新したいという意思だけはあるのですが、本人の性格上の問題（=めんどくさがり）により、極めて更新頻度が低くなっております。Live 告知なども場合によっては本番直前に更新されたりしますので、何のために個人サイトを運営してるんだかよく分かりませんが、それでも「何もせんよりはアップするだけマシや」の精神で、今後も細々と続けていきたいと思っております。

### ハイパーリンクについて &mdash; リンクはご自由にどうぞ {#Hyperlink}

著者は「Web 上にあるリソースは、ハイパーリンクを介して自由に移動できてこそ、初めてその威力が発揮される」と考えております。

ただし、その際に例えば

```html

```


のようなアンカーテキスト（リンクテキスト）を利用されますと、スクリーンリーダーを使用するユーザーが利用しづらくなる、文書を印刷して読む場合に意味不明になる、などの問題が発生することが指摘されています。

アンカーテキストには単独で何のページなのか分かる、具体的な内容を書くのが賢明です。できればページのタイトル（`title` 要素の内容）を使用していただけるとありがたいです。

#### 埋め込みコンテンツとして利用される場合（いわゆる「直リン」について） {#Embedding}

他のサイトにはハイパーリンクとこの「直リン」をどちらも「リンク」として取り扱い、例えば「リンクは自由ですが直リンは禁止です」のような規約を設けているところもありますが、当サイトでは「リンク」= ハイパーリンクのことであり、いわゆる「直リン」は<dfn>埋め込みコンテンツ</dfn>として、リンクとは別ものとして扱っております。

よって、当サイトのコンテンツを埋め込みコンテンツとして利用される場合には、[ハイパーリンクの項](#Hyperlink)で述べた条件は適用されませんのでご注意ください。

### Cookie（クッキー）について {#Cookie}

<ruby><dfn>Cookie</dfn><rp>（</rp><rt>クッキー</rt><rp>）</rp></ruby> とは、ユーザーが Web サイトを訪れた時に、Web ブラウザを介してそのユーザーが使用するデバイスに保存される、小さなテキストデータのことです。現在、当サイトが直接的にセットしている Cookie はございませんが、当サイトで利用中の次のツールおよびサービスが Cookie を使用して、閲覧履歴や利用状況などの情報を収集しています。

- アクセス解析ツール（Google アナリティクス）
- 広告配信サービス（Google AdSense）

### 広告について {#Ads}

当サイトのほぼすべてのページには、以下の広告事業者によるいずれか一つ、もしくは複数の広告が掲載されています。

#### Google AdSense {#AdSense}

当サイトは、Google が提供する広告配信サービスである <dfn>Google AdSense</dfn> を利用しております。

この Google AdSense は広告の配信に Cookie を使用しています。Cookie はユーザーが Google のパートナー Web サイトを訪問するとセットされ、Google および当サイトはこの Cookie を使用してユーザーの利用デバイスを識別しますが、個人を特定できる情報そのものは含まれません。

Google などの第三者配信事業者は Cookie を使用し、ユーザーが過去にアクセスした Web サイトの情報に基づいて広告を配信しています。また Google は広告 Cookie を使用することにより、個々のユーザーの興味に応じた最適な広告（パーソナライズド広告）を Web サイトに表示しています。

このようなパーソナライズド広告をお望みでない場合は、Google の「[マイ アド センター][myadcenter]」でオフにすることができます。また、第三者配信事業者がパーソナライズド広告の掲載で使用する Cookie を無効にしたい場合は、[optout.aboutads.info][optout] にアクセスすることで、事業者ごとにオプトアウトすることができます。
